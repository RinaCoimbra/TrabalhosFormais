----------------------------------------------------------------
  1. Compilando
----------------------------------------------------------------

	Para compilar o LEXICO, execute o comando:
		make lexico

	Em um terminal UNIX.

----------------------------------------------------------------
  2. Executando
----------------------------------------------------------------

	Para executar o LEXICO, execute o comando em um terminal UNIX:
		make lrun < arquivodetestes.awk

	O arquivodetestes.awk do exemplo deverá conter um programa na
linguagem AWK.
