----------------------------------------------------------------
  1. Compilando
----------------------------------------------------------------

	Para compilar o EXTRATOR, execute o comando:
		make compile

	Em um terminal UNIX.

----------------------------------------------------------------
  2. Executando
----------------------------------------------------------------

	Para executar o EXTRATOR, execute o comando:
		make run

	Em um terminal UNIX.

	A saída pode ser encontrada no arquivo extraction.txt
